# pd-reach-grasp-cortical-subcortical
EEGLAB- and MNE-based pipeline for analyzing cortical–subcortical interactions during reach and grasp movements in Parkinson’s disease


## Repository structure

- preprocessing/: preprocessing scripts
- time_frequency_analysis/: LFP and EEG time–frequency analyses
- spectral_statistical_analysis/: group-level CBPT and subject-level statistical analyses
- coherence_analysis/: imaginary coherence analyses
- granger_causality_analysis/: spectral Granger causality analyses
- bootstrap_testing/: bootstrap testing for connectivity


## Dependencies

### Python
Tested with **Python 3.10.18**.

### MATLAB
Tested with **MATLAB R2024a**. Required toolboxes:
- **EEGLAB** (v2025)
